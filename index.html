Vamos a comenzar escribiendo las partes basicas para hacer que nuestra pagina html funcione, el cual dividiremos en pasos:

PASO 1: Primero vamos a empezar escribiendo (html) para indicar que sera una pagina html.

PASO 2: Para poder ponerle un nombre a la pagina se debera poner (head), de ahi pondremos (title) y enseguida escribimos el nombre de como se va a llamar el documento. (Para hacer validas estas funciones tendremos que cerrar el comando con una "/" antes del comando, el cual seria: (/title) eso es para indicar que hasta ahi llega el nombre de nuestra pagina, para cerrar el titulo y darle la indicacion que ya hemos escrito como se llamara nuestra pagina o documento escribimos (/head).

PASO 3: Para tener un cuerpo de informacion pasaremos a escribir (body), el cual puede incluir titulo, texto e imagenes.

PASO 4: Una vez llegando a este paso es el momento de poner nuestro titulo, pondremos este comando "h1" para indicar que ese sera el titulo de lo que queremos hablar o dar a conocer, ponemos (h1)TITULO(/h1), siempre que queremos marcar o indicar que hasta ahi llega el comando tendremos que cerrarlo con "/comando".

PASO 5: Para poner un texto, usaremos el comando "p": (p), cuando terminemos de poner el texto, lo cerraremos con (/p)

PASO 6: Si queremos agregar una imagen, pondremos el siguiente codigo: (img src="./nombre de la foto.png" /), los siguientes pasos son explicando el comando y lo que se debe poner.

S1: Escribimos img scr= para indicar que pondremos una imagen.

Para poder centrar un texto o imagen, el comando es (align="center"), y si le quieres dar justificacion a tu texto, el comando es (align="justify")

PASO 7: Para cerrar el cuerpo de la pagina, escribiremos lo siguiente (/body).

PASO 8: Llegamos al final, para indicar que ya acabamos con todo lo que queremos mostrar en nuestra pagina, cerraremos con el comando (/html).
